# ts-node 패키지

타입스크립트 코드를 별도의 컴파일 없이 명령어로 즉시 실행할 수 있게 한다.

REPL도 지원

# 명령어 예시
```
npx ts-node --project tsconfig.json ./src/index.ts 
```
* --project tsconfig.json 프로젝트
* ./src/index.ts 진입점
