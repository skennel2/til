# 라이프타임 
보통 라이프타임은 암묵적으로 추론된다. (대부분의 타입이 추론되어 명시하지 않아도 되듯)
라이프타임은 참조자의 수명을 명시할수 있는 방법이다.

# 대여 검사기

```rust
fn main() {
    let r;                // ---------+-- 'a
                          //          |
    {                     //          |
        let x = 5;        // -+-- 'b  |
        r = &x;           //  |       |
    }                     // -+       |
                          //          |
    println!("r: {}", r); //          |
} 
```
